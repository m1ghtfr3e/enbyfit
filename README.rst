==============================================
Enbyfit: Fitness and Health Modules for Humans
==============================================


Fitness App for Non-Binaries, though for some equations it is still needed to specify a hormonal sex.


Installation
------------
- **Via PyPi**
  Download and install it using pip::
  ``` 
  pip install enbyfit 
  ```
